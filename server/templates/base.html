The EGA Beacon CLI allows you to find any specific genomic variant in a region of a chromosome. To get some results, it is compulsory to fill in the form with the reference of the genome, the number of the chromosome you want to query at and a start position inside the region where you want to find the genomic variation. Optionally, you can also add some specific mutation allele you want to find and if the data has any liftover or public condition. To specify how you want to get the result, you have to select BOOL (for Yes or No answer) or COUNT (for the number of genomic variations found for your query).

TRUE

A true query will return some results found for a specific reference genome, chromosome and start position. Click on Example to fill the form with a true query.

Example
FALSE

A false query will struggle to find any result in a specific reference genome, chromosome and start position. Click on Example to fill the form with a false query.
